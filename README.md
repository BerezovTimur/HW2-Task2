## Отчёт о тестировании приложения "Precision"

### Краткое описание

На основе предложенного кода было создано приложение "Precision". Приложение выдает сумму итоговых бонусов начисленных клиенту.
Переменные используемы в приложении:
- regularBonus - обычный бонус;
- specialBonus - специальный бонус;
- totalBonus - итоговый бонус.

### Описание использованных тестов

При тестировании приложения использовалось функциональное тестирование методом эквивалетных значений.
Исходя из исходных данных были взяты следующие данные: 0,3 и 0,6
